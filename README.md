## 🚀 소개
FlimChoice는 HTML, CSS, JavaScript를 사용하여 처음으로 개발한 영화 정보 웹사이트입니다. 이 웹사이트는 사용자가 영화를 검색하고, 정보를 확인할 수 있도록 설계되었습니다.

## ✨ 주요 기능
- **영화 검색**: 영화 제목을 입력하여 검색
- **영화 상세 정보**: 각 영화에 대한 자세한 정보를 제공
- **영화 예매**: 선택한 영화관 예매 사이트로 이동
